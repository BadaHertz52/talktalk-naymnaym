# 톡톡냠냠 (talktalk-naymnaym)

감정을 적으면 당근이 굴러와 먹어치우는 코어 루프 웹앱.

## 스택
Vite + React + TypeScript / Zustand / react-router / Canvas

## 로컬 실행
```bash
nvm use          # Node 22 (.nvmrc)
pnpm install
pnpm dev
```

## 스크립트
| 명령 | 설명 |
|---|---|
| `pnpm dev` | 개발 서버 |
| `pnpm build` | 프로덕션 빌드 (`dist/`) |
| `pnpm lint` | oxlint |
| `pnpm type-check` | `tsc -b` 타입 검사 |
| `pnpm test` | vitest |
